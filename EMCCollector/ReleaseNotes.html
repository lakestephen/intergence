Release Notes for EMC Collector

Release Date Notes
emccollector-1.0.zip 6 April 2014 Displays all the disks in the system on HyperGlance, but without any formal structure. Server ip addresses/port/user/passwords are all hard coded at the moment.
emccollector-1.1.zip 8 April 2014 Added the central node, but is not properly named yet.
All disks and links are in place, but no disk used capacity yet.
emccollector-1.2.zip 10 April 2014 Includes all the basic structure in the model.
Server ip addresses/port/user/passwords are all hard coded at the moment, but this will be moved out into config for the next release.
emccollector-1.3.zip 13 April 2014 Move all config into config.properties file.
emccollector-1.4.zip 25 April 2014 Fix connection issues with previous release.
Add control over the threads that can connect to EMC.
Changed packaging to include release notes.
