Release Notes for Vmware Rest Collector

Release Date Notes
VmwareRest-0.1.zip 30 January 2015 Initial Release.
At the moment the collector is not writing to hyperglance, but logging actions through the common framework that I have set up.
VmwareRest-0.2.zip 4 February 2015 Added Linux .sh scripts for starting.
Target Java verison = 1.6.
VmwareRestCollector-0.3.zip 9 February 2015 Correct issue with properties file being supplied in the jar and separately. The one in the jar was overriding any changes to the esposed one.
VmwareRestCollector-0.4.zip 7 April 2015 Work with standard rest framework.
VmwareRestCollector-0.5.zip 12 April 2015 Add support for creating and dropping the credentials database.
Add better logging of unhandled exceptions
Fix bug where the node and endpoint were swapped.
Increase Memory to 128M
Ensure duplicate topology can't be added.
VmwareRestCollector-0.6.zip 13 April 2015 Add request timeout to calls to the HG rest service.
VmwareRestCollector-0.7 15 September 2015 Transition to merged processing, allowing linking of the nodes from the different models even when using the limited functionality of the Rest API V1.0.
